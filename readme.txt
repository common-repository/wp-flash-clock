== Description ==

WP-Flash Clock Widget gives you the ability to add a flash based clock to your Wordpress blog. There are 15 different versions of the clock that you can install on your blog.


== Installation ==

1. Download the zip file and extract the contents.
2. Make sure you're running WordPress version 2.3 or better. It won't work with older versions.
3. Upload the 'wp-flashclock-widget' folder to your plugins directory (plugins/wp-flashclock-widget).
4. Activate the plugin through the 'plugins' page in WP.
5. See widget options to adjust things like display size, clock style etc...


== Frequently Asked Questions ==

= My theme/blog won't show the flash clock correctly.Why is this? =

- Make sure your website validates correctly. Some markup errors can cause issues when the WP FlashClock widget is used.

- Please make sure you have at least version 9 of Flash player and javascript enabled in your browser.


= How Do I Change The Clock Face =

It's easy to change the style of clock you want to use. When you add the widget it will ask you which clock number you would like to display. You can enter numbers 1-15 to change the style.

= How Do I Change The Size Of The Clock =

It's easy to change the size of the clock also. Simply change the settings on the add widget page to the size that you prefer.
